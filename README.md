# MAX Video Circle Bot

Этот бот для мессенджера MAX помогает вам превратить обычные видео или GIF-анимации в видеокружки.

## ✨ Возможности

- **Конвертация видео и GIF:** Просто отправьте боту видеофайл или GIF, и он превратит его в квадратный видеокружок.
- **Обрезка длинных видео:** Если ваше видео длиннее 60 секунд, бот предложит обрезать его.
- **Управление звуком:** Вы можете выбрать, оставить или удалить звуковую дорожку из видео.
- **Автоматическое форматирование:** Бот автоматически обрезает видео до квадрата из центра, чтобы оно идеально подходило для формата кружка.

## 🔑 Как получить токен

Для работы бота в мессенджере MAX вам понадобится API-токен. Получить его можно двумя способами:

1.  **Через MasterBot:**
    - Найдите в мессенджере MAX бота с именем **MasterBot**.
    - Следуйте инструкциям для создания нового бота (аналогично BotFather в Telegram).
    - После создания вы получите токен.

2.  **Через партнерский портал:**
    - Зайдите на портал [max.ru](https://max.ru) (или соответствующий корпоративный портал вашей организации).
    - Перейдите в раздел управления ботами.
    - Выберите вашего бота и в разделе "Интеграция" скопируйте токен.

## 🚀 Как запустить

### С помощью Docker (рекомендуется)

1.  **Склонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/max-video-circle-bot.git
    cd max-video-circle-bot
    ```

2.  **Создайте файл `.env`:**
    Скопируйте `.env.example` в `.env` и впишите свои учетные данные для входа в MAX.
    ```bash
    cp .env.example .env
    ```
    Затем отредактируйте `.env`:
    ```
    LOGIN=YOUR_LOGIN
    TOKEN=YOUR_TOKEN
    ```

3.  **Запустите бота с помощью Docker Compose:**
    ```bash
    docker-compose up -d
    ```

### Локальный запуск

1.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Настройте учетные данные:**
    Отредактируйте файл `config.py` и впишите свои `LOGIN` и `TOKEN`.

3.  **Запустите бота:**
    ```bash
    python main.py
    ```

## ⚙️ Принцип работы

1.  **Получение медиа:** Бот принимает видеофайлы и GIF-анимации.
2.  **Анализ:** Он проверяет длительность медиа.
3.  **Опции:** Бот предлагает опции: обрезать видео (если оно слишком длинное) и/или удалить звук.
4.  **Конвертация:** С помощью библиотеки `moviepy` бот выполняет следующие действия:
    - Обрезает видео до 60 секунд (при необходимости).
    - Удаляет звук (при необходимости).
    - Изменяет размер и обрезает видео до квадрата 360x360 пикселей.
    - Кодирует видео в формат MP4, совместимый с MAX.
5.  **Отправка:** Готовый видеокружок отправляется обратно пользователю.
